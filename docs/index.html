ThermalSystems_Models.jl Documentation

This package was developed for modeling multi-domain physical systems. The package consists of several block libraries, as well as simplified models for faster computation. This documentation will guide you through the features and usage of the library.

Features

  • Multi-domain fluid thermal systems (currently supports multiphase, gasous, and incompressible liquids)
  • Dynamic and steady state models
  • Optionality for complex vs simple models
  • Visualization tools for block structures
